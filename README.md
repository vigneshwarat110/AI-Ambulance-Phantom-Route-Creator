# 🚑 AI Ambulance Phantom Route Creator

Peak-hour congestion in Tambaram delays ambulances by up to **47 minutes**, leading to preventable deaths.  
The **AI Ambulance Phantom Route Creator** uses GPS tracking and real-time AI traffic prediction to detect gridlock and dynamically synchronize multiple traffic signals to form temporary multi-lane emergency corridors.  

A Firebase-powered AR dashboard tracks ambulance movement, routes, and time saved in real time.

---

## 📌 Problem Statement

Tambaram experiences severe peak-hour congestion, significantly delaying emergency vehicles. Traditional siren-based clearing is ineffective in high-density traffic, causing:

- Extended ambulance travel times
- Increased mortality risk
- Inefficient manual signal management
- Lack of real-time monitoring

---

## 🎯 Solution

The AI Ambulance Phantom Route Creator:

1. Tracks ambulances using live GPS.
2. Predicts congestion using AI-based traffic analysis.
3. Detects gridlock zones in advance.
4. Overrides and synchronizes multiple traffic signals.
5. Creates a temporary “Phantom Route” corridor.
6. Displays everything in a Firebase AR dashboard.

---

## ✨ Key Features

- 🚑 Real-time Ambulance GPS Tracking  
- 🤖 AI-Based Traffic Prediction  
- 🚦 Dynamic Multi-Signal Synchronization  
- 🛣 Temporary Multi-Lane Emergency Corridor  
- 📊 Firebase AR Monitoring Dashboard
